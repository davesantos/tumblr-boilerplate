Tumblr Boilerplate

A fully functional bare-bones Tumblr theme that works out of the box.

Style it to your needs. The goal of the project was to remove uncessary code easing the development process.

  • Download 2.0
  • Source

Features

Supports all post-types

All essential post-types (Image, Video, Text, Link, etc.) are semantically structured

Localization Strings

Select variables that appear in other Tumblr supported languages.

Normalize CDN

The widely adopted CSS Normalize minified & hosted on a realiable delivery network.
